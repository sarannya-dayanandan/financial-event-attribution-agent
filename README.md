# financial-event-attribution-agent
A financial event attribution platform that explains market movements by linking asset price changes to macroeconomic and geopolitical events.

## Problem

Most financial dashboards tell users what happened.

Very few explain why it happened.

This project aims to answer:

- What changed?
- Why did it change?
- Which events contributed?
- What evidence supports the explanation?

## Assets Covered

### Precious Metals
- Gold
- Silver

### Foreign Exchange
- USD/INR
- EUR/INR
- GBP/INR

### Supporting Indicators
- DXY
- US 10Y Treasury Yield
- Crude Oil
